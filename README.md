# Chat-Application
chat Application built using Nodejs, Reactjs, socket.io
## Installation

Server
```bash
cd server
```
```bash
npm install
```
```bash
npm start
```
Client

```bash
cd client
```

```bash
npm install
```
```bash
npm start
```

## Deployment
Server deployed  on vercel and Client deployed on netlify
